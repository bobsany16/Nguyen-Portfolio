Tip Calculator

Fubuki Puzzle Solver

The Fubuki Puzzle Solver helps computing the solutions to Fubuki Puzzles (as shown above). User is prompted to enter the position (a1 - a9), its value of 1-9, and the corresponding R and C values.
The program will output the solutions along with the total number of solutions.
This program is coded using Python Dash, later translated into R Shiny. Implemented the use of NumPy arrays and permutation function to find the numbers that will match the R and C values while fit with the position and valued selected.
